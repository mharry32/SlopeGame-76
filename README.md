# SlopeGame-76

A browser-based Slope game project. This repository contains the static site files (HTML, JavaScript, assets) needed to run the game locally or host it on a web server.

## Live build (optional)
If you want a hosted version for quick preview/testing:

https://slopeunblocked76.net

> This link is provided as a convenience for previewing the current build.

---

## Repository contents (high level)
- Main entry page: `index.html`
- Additional pages: `frame.html`, `ubg235.html`, `games235.html`, `404.html`, `408.html`
- Game/runtime scripts: `game-scripts.js`, `loading__.js`, `modules__.js`, `settings__.js`, `start__.js`
- Runtime bundle: `playcanvas-stable.min.js`
- Styling: `styles.css`
- Web app files: `manifest.json`, `appmanifest.json`, `sw.js`
- Folders (assets/data/other): `assets/`, `files/`, `js/`, `json/`, `patch/`, `tmp/`

---

## Run locally

### open in a browser
1. Download or clone the repository
2. Open `index.html` in your browser
